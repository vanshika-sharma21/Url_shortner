# 🔗 URL Shortener

A full-stack URL Shortener application that converts long, complex URLs into short, easy-to-share links while providing tracking and analytics for better link management.

---

## 📌 Problem Statement

Long URLs are often difficult to share, remember, and manage—especially on platforms like social media, emails, and messaging apps where space and readability matter. Additionally, there is no easy way to track how many users interact with a shared link.

---

## 💡 Solution

This project solves the problem by:
- Converting long URLs into short, clean links
- Making links easy to share and manage
- Providing tracking and analytics for each URL
- Allowing users to monitor engagement and usage

---

## 🚀 What This Project Does

- Takes a long URL as input and generates a unique short code  
- Redirects users from short URL → original URL instantly  
- Stores user data and URLs securely  
- Tracks number of clicks for each shortened link  
- Provides a dashboard to manage and monitor URLs  
- Supports authentication using JWT and OAuth  

---

## 🎯 Target Users

- Students sharing notes/resources  
- Developers sharing project/demo links  
- Content creators & marketers tracking engagement  
- Anyone who wants clean and short links  

---

## 🚀 Features

- 🔗 Shorten long URLs instantly  
- 👤 User authentication (JWT & OAuth)  
- 📊 Click tracking & basic analytics  
- 🔄 Fast redirection system  
- 🧾 User dashboard for managing links  
- 🔐 Secure login & protected routes  

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MySQL  
- **Authentication:** JWT, OAuth (Google/GitHub)
